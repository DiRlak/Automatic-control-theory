# Automatic-control-theory
Репозиторий с лабораторными работами по "Теории систем автоматического управления"

Это учебные материалы, выполненные в рамках курса по теории систем автоматического управления, за авторством преподавателя университета МИСИС Тихонова Андрея Ильича.
Используйте информацию из этого репозитория как справочный материал, а не для прямого цитирования.

## Список лабораторных работ:
1.  ЛР 01. Решение обыкновенного дифференциального уравнения с использованием структурной модели
2.  ЛР 02. Построение структурной модели исполнительного двигателя постоянного тока (ДПТ) с постоянными магнитами
3.  ЛР 03. Построение структурной модели генератора постоянного тока (ГПТ) с независимым возбуждением
4.  ЛР 04. Построение структурной модели САР напряжения ГПТ 
5.  ЛР 05. Исследование временных и частотных характеристик элементарных динамических звеньев
6.  ЛР 06. Построение частотных характеристик разомкнутой одноконтурной САР ГПТ
7.  ЛР 07. Законы регулирования на примере САР ГПТ
8.  ЛР 08. Исследование на устойчивость САР ГПТ
9.  ЛР 09. Качество САУ
10. ЛР 10. Синтез САУ
11. ЛР 11. Моделирование САУ в пространстве состояний
